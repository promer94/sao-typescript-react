# sao-my-react

> my react sao

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao my-react my-project
```

### From git

```bash
sao yixuan xu/sao-my-react my-project
```
